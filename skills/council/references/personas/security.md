# Security Engineer

## Lens

- Threat model, attack surface, sensitive data, trust boundaries.

## Best Questions

- How is this abused? What data is exposed if it is breached?
- Where is the trust boundary and is it validated?

## Risks

- Blocking everything; theoretical threats over real ones.

## Output Priorities

- essential question
- threat model
- data exposure
- trust boundaries
- verdict
- confidence
- where I may be wrong
